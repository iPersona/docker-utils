# Docker Utils
A tool set to make docker easier to use.

## How to use

- Clone the repo.

```
git clone https://github.com/iPersona/docker-utils.git docker-utils
```

- cd to the folder

```
cd docker-utils
```

- Change permission

```
chmod 755 *
```

- Enjoy!

## Tool Set
### du_clean.sh
Remove unused container and images.
